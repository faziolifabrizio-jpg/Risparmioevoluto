
# Amazon Telegram Bot – Scraping Avanzato

Questo bot estrae automaticamente le offerte Amazon GoldBox e le pubblica sul tuo canale Telegram **alle 09:00 e alle 21:00** ogni giorno.

## INSTALLAZIONE

1. Carica questi file su GitHub.
2. Vai su:
   Settings → Secrets → Actions
3. Aggiungi:
   - TELEGRAM_TOKEN
   - TELEGRAM_CHAT_ID
4. Attiva Actions → Run workflow

Tutto funziona automaticamente.
